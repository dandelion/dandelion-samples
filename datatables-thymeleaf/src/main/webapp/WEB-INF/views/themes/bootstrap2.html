Bootstrap 2 theme

In order to activate the Bootstrap 2 theme, just :

  • set the dt:theme table attribute to bootstrap2
  • add the table Bootstrap CSS classes using cssClass table attribute depending on your needs

In a nutshell, you need to include in your page the following assets :

  • jQuery of course :-)
  • Bootstrap (JS + CSS)
  • DataTables (JS + CSS + IMG)
